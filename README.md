# Tic-Tac-Toe-Game
TicTacToe is a simple and interactive two-player web game developed using HTML, CSS, and JavaScript. The game allows two players to play alternately as X and O on a 3×3 grid. It includes winner detection, draw handling, and game reset functionality with a clean and responsive user interface.

Features
Two-player gameplay (X and O)
Automatic winner detection
Draw game detection
Reset and New Game options
Interactive and responsive design
Dynamic winner announcement message
Technologies Used
HTML for creating the game structure
CSS for styling and responsive layout
JavaScript for game logic and interactivity
How the Game Works
Players take turns clicking on empty boxes.
Player O starts the game, followed by Player X.
The game checks for winning combinations after every move.
If a player matches three symbols in a row, column, or diagonal, they win.
If all boxes are filled without a winner, the game ends in a draw.
Players can restart the match using the Reset Game or New Game buttons.

This project is perfect for beginners to practice DOM manipulation, event handling, arrays, loops, conditional statements, and game logic in JavaScript while building a fun and engaging application.
